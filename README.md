# Real Estate Manager

**Real Estate Manager** — десктопное приложение для Windows, предназначенное для управления квартирами и бронированиями. Позволяет риэлторам регистрировать пользователей, добавлять/удалять квартиры, просматривать бронирования и фото недвижимости. Разработано в рамках учебного проекта с использованием Python и PyQt5 для удобного интерфейса.

Приложение использует базу данных SQLite для хранения данных о пользователях, квартирах и бронированиях.

---

## Возможности

- Регистрация и вход пользователей (администратор/риэлтор)
- Добавление, редактирование и удаление квартир (цена, площадь, комнаты, адрес, этаж, фото)
- Просмотр списка квартир с деталями и фото
- Управление бронированиями (просмотр, добавление, удаление)
- Поиск квартир по критериям (цена, комнаты, этаж)
- Уведомления о просроченных бронированиях
- Адаптивный интерфейс с изменением размеров элементов

## Установка и запуск

### Требования
- Python 3.8+
- Установленный PyQt5 (`pip install PyQt5`)
- Windows 10+

### Шаги по установке

1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/RascallySine18/realme-estate-app.git
2. Установите зависимости:
   ```bash
   pip install PyQt5
3. Запустите приложение:
   ```bash
   python main.py
